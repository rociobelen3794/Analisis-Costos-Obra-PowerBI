# Análisis de Costos de Obra - Power BI

Dashboard interactivo desarrollado en Power BI para analizar la estructura de costos de un proyecto de construcción.

## Objetivo

Analizar cómo se distribuyen los costos de una obra y detectar qué rubros tienen mayor impacto en el presupuesto total

## Herramientas utilizadas

- Power BI
- Modelado de datos
- DAX
- Visualización de datos

## Análisis realizados

El dashboard incluye:

- El costo total del proyecto
- Distribución de costos por rubro
- Comparación entre material y mano de obra
- Identificación de tareas con mayor incidencia en el costo
- Análisis de Pareto para detectar los rubros que explican el 80% del costo total

## Principales hallazgos

- Un número reducido de rubros concentra la mayor parte del costo total.
- La estructura y las instalaciones representan una proporción significativa del presupuesto.
- El análisis de Pareto permite identificar los componentes críticos del costo de obra.
